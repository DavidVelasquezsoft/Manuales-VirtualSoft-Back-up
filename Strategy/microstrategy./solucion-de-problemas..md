# Solución de problemas.

### Errores Comunes

Aquí se listan los problemas más frecuentes y sus posibles soluciones.

| Problema                                               | Posible Causa                                               | Solución                                                                                                                      |
| ------------------------------------------------------ | ----------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| Los informes tardan demasiado en ejecutarse            | Consultas complejas o sobrecarga del servidor               | Optimiza los filtros y reduce la cantidad de datos procesados. Consulta con el equipo de TI si el servidor está sobrecargado. |
| No puedo iniciar sesión en MicroStrategy               | Credenciales incorrectas o cuenta bloqueada                 | Verifica el usuario y contraseña. Si el problema persiste, restablece la contraseña o contacta al administrador.              |
| Error al conectarse a la base de datos                 | Configuración incorrecta o problemas de red                 | Verifica la configuración de la conexión y asegúrate de que la base de datos esté accesible.                                  |
| No se muestran los datos esperados en el reporte       | Filtros incorrectos o permisos insuficientes                | Revisa los filtros aplicados y valida que tienes acceso a los datos requeridos.                                               |
| Error en la generación de documentos o tableros        | Incompatibilidad de formato o datos faltantes               | Asegúrate de que las fuentes de datos sean correctas y prueba con otro formato de salida.                                     |
| Problema con la actualización de datos en un Dashboard | Fuente de datos desconectada o errores en la sincronización | Verifica que la conexión a la fuente de datos esté activa y actualiza manualmente si es necesario.                            |
