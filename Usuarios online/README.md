# Usuarios online

### **Introducción**

Este manual de usuario ha sido creado para orientarte en el uso de la plataforma. Aquí encontrarás instrucciones detalladas sobre cómo acceder y utilizar cada una de las secciones disponibles, facilitando así una experiencia completa y eficiente.

### **Objetivo**

Brindar una guía clara, práctica y accesible para los usuarios de la plataforma, con el fin de que puedan aprovechar al máximo todas las herramientas, funciones y servicios que esta ofrece.
