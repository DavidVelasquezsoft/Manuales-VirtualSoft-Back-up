# Gestión

En el submenú de Gestión puedes administrar tu cuenta, ajustar preferencias de juego y acceder a informes detallados. También puedes establecer límites de apuestas y tiempo para un juego responsable, selecciona la opción correspondiente para acceder a la información detallada.

### Configuración general

#### 1. Acceso al Módulo:

**Ruta de Acceso**: Menú principal → Gestión.
