# Póker

{% hint style="warning" %}
**Nota:** Este apartado se encuentra disponible únicamente en determinados países en la plataforma doradobet.
{% endhint %}

***

## 1. Acceso al Módulo <a href="#id-1.-acceso-al-modulo" id="id-1.-acceso-al-modulo"></a>

**Ruta de Acceso**: Menú principal > Póker

***

## 2. Flujo para acceso a Póker Playtech

{% stepper %}
{% step %}
### Iniciar sesion

Inicia sesion en la plataforma una vez ya estes registrado.

Guia completa aqui: [Registo](registro.md) | [Inicio de sesión](inicio-de-sesion/)
{% endstep %}

{% step %}
### Desplegar barra lateral

Una vez dentro, despliega la barra lateral izquierda en la plataforma.
{% endstep %}

{% step %}
### Elige Póker

se despegará una lista de opciones para ingresar a diferentes juegos, en este caso, debes ingresar por la sección "**POKER**".
{% endstep %}

{% step %}
### Ingresa a Póker Playtech

En este login debes ingresar tú mismo usuario y contraseña de tu cuenta principal de la plataforma.

Al iniciar sesión, el sistema realizará automáticamente la apertura de tu cuenta en ambas plataformas de forma simultánea.

Esto permitirá que, dentro de la plataforma de **Playtech**, se refleje El mismo saldo disponible que en la plataforma, habilitándote para realizar apuestas directamente desde Playtech sin necesidad de transferencias manuales.
{% endstep %}
{% endstepper %}
