### **3. Validaciones y Reglas de Negocio**

* Solo se permite un registro por persona y por dispositivo.
* Todos los retiros requieren validación de identidad.
* Los bonos tienen requisitos de apuesta (rollover) definidos en los Términos y Condiciones.
* La plataforma impide el juego a menores de edad.
* Cuando el usuario sea desverificado en la plataforma, al intentar realizar acciones restringidas (según país o partner) como depositar o retirar se mostrará un mensaje de verificación requerida con las indicaciones para recuperarla en [Verificar cuenta](gestion/verificar-cuenta..md). (_Ej: actualizar documento_)
*   Cuando el usuario este desverificado e intenta realizar un **retiro**, se mostrará el siguiente mensaje:

    > **“Tu documento de identidad está vencido y tu cuenta ha sido desverificada. Debes actualizar tu documento para volver a verificarte y poder realizar retiros.”**
*   Cuando el usuario este desverificado e intenta realizar un **depósito**, se mostrará el siguiente mensaje:

    > **“Tu documento de identidad está vencido y tu cuenta ha sido desverificada. Debes actualizar tu documento para poder realizar depósitos.”**
