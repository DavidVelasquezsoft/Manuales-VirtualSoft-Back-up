# Solución de problemas.

### Errores Comunes

Aquí se listan los problemas más frecuentes y sus posibles soluciones.

| **Problema**                   | **Posible Causa**                                  | **Posible Solución**                                                                                          |
| ------------------------------ | -------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| No puedo iniciar sesión.       | Usuario o contraseña incorrectos.                  | Verifica los datos ingresados e intenta recuperar la contraseña si es necesario.                              |
| No se procesa el retiro.       | Cuenta no verificada o saldo insuficiente.         | Asegúrate de haber verificado tu cuenta y revisa que el saldo disponible cumpla con los requisitos de retiro. |
| La apuesta no se registra.     | Problema de conexión o error en la selección.      | Revisa tu conexión a Internet y verifica que la apuesta cumpla con los términos y condiciones.                |
| Método de pago rechazado.      | Datos incorrectos o método no disponible.          | Confirma que los datos ingresados sean correctos e intenta con otro método de pago.                           |
| Cuenta suspendida o bloqueada. | Incumplimiento de términos o actividad sospechosa. | Contacta con el soporte de Doradobet para obtener más información y resolver el problema.                     |
| Error registro de la cuenta.   | Usuarios prohibidos.                               | Verificar el correo electrónico donde se detalla el motivo de la prohibición.                                 |
| Banner cambio de contraseña.   | Se ha expirado la contraseña de la cuenta.         | Realizar cambio de la contraseña.                                                                             |
