# Crear Bono.

Para comenzar, selecciona la opción "**Crear bono**".

Al seleccionar esta opción, accederás al apartado destinado para la creación de un bono. Como primer paso, deberás seleccionar el país para el cual deseas generar el bono.

<figure><img src="../../../../.gitbook/assets/image (132).png" alt="" width="212"><figcaption><p>Figura #1: Captura de pantalla país bono.</p></figcaption></figure>

Luego de seleccionar el país, podrás escoger el tipo de bono que deseas crear.

<figure><img src="../../../../.gitbook/assets/image (133).png" alt=""><figcaption><p>Figura #2: Captura de pantalla tipos de bonos.</p></figcaption></figure>

Para más información sobre cómo crear el tipo de bono de tu preferencia, puedes seleccionar la página correspondiente.
