# Flujos de Creación de Torneos

Existe un único tipo de torneo. No obstante, el proceso de creación puede variar según el programa o campaña correspondiente.

En cada caso se define un flujo específico que determina:

* Campos requeridos.
* Validaciones aplicables.
* Configuraciones adicionales, si corresponde.

A continuación, se presenta el listado de manuales donde se detalla paso a paso cada flujo disponible.
