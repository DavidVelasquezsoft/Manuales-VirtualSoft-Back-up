#### 4.1. Añadir producto partner

Abre pop-up que permite agregar un producto a un partner indicando  los siguientes campos.

> Para más información sobre cómo y dónde crear productos, consulte el [manual de creación de productos](../../../productos/productos.md#id-3.1.-anadir-producto).

<table><thead><tr><th width="155.3333740234375">Campo</th><th>Descripción</th></tr></thead><tbody><tr><td><strong><code>Productos</code></strong></td><td>Seleccione un producto previamente creado para asignarlo al partner. </td></tr><tr><td><strong><code>Partner</code></strong></td><td>Indique el partner al que se asociará el producto.</td></tr><tr><td><strong><code>Estado</code></strong></td><td>Define si el producto estará activo o inactivo para el partner.</td></tr><tr><td><strong><code>Verificado</code></strong></td><td>Indica si el producto ha sido validado para su uso.</td></tr><tr><td><a producto.</td></tr><tr><td><strong><code>Tiempo de Procesamiento</code></strong></td><td>Especifica el tiempo estimado para la gestión de operaciones del producto.</td></tr><tr><td><strong><code>Máximo</code></strong></td><td>Define el valor máximo permitido para las transacciones.</td></tr><tr><td><strong><code>Mínimo</code></strong></td><td>Establece el valor mínimo permitido para las transacciones.</td></tr><tr><td><strong><code>Orden</code></strong></td><td>Determina la posición en la que se mostrará el producto en la
