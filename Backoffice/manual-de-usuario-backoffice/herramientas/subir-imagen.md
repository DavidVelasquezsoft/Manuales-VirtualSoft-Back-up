# Subir imagen

{% hint style="warning" %}
**Nota:** El acceso al módulo requiere permisos específicos; en caso de no contar con ellos, no será visible en el sistema.
{% endhint %}

**Ruta de Acceso**: BackOffice (BO) > Herramientas > Subir Imagen

### 4. Subir Imagen

Permite cargar una imagen al sistema. Para ello:

1. Selecciona el botón **Subir Imagen**.
2. En la ventana emergente, elige el archivo de imagen desde tu dispositivo.
3. Confirma la selección para iniciar la carga.

{% hint style="warning" %}
**Nota:** El archivo debe cumplir con los formatos permitidos: **JPG, PNG o GIF**.
{% endhint %}

***

### 5. Obtener URL

Una vez cargada la imagen, el botón **Obtener URL** abre una ventana emergente con el resultado del proceso:

<table><thead><tr><th width="180">Resultado</th><th>Descripción</th></tr></thead><tbody><tr><td><strong>Carga exitosa</strong></td><td>La ventana emergente muestra la URL generada para la imagen. Esta puede copiarse desde el campo habilitado para utilizarla en otros apartados de la plataforma.</td></tr><tr><td><strong>Carga fallida</strong></td><td>La ventana emergente muestra el mensaje de error correspondiente, indicando que la imagen no pudo cargarse.</td></tr></tbody></table>

***

### 6. Validaciones y reglas del negocio:

* El acceso al módulo requiere permisos específicos; de lo contrario, no será visible en el sistema.
* Solo se permiten imágenes en los formatos JPG, PNG y GIF.
* La URL únicamente se genera cuando la imagen se carga correctamente; si la carga falla, se muestra el mensaje de error correspondiente.
