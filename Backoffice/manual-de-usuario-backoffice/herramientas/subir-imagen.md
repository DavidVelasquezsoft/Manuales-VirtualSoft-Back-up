# Subir imagen

## **Subir una imagen:**

* Haz clic en el botón **"Subir Imagen"**.
* Se abrirá una ventana emergente donde podrás seleccionar un archivo desde tu dispositivo.
* Asegúrate de que el archivo cumpla con los formatos permitidos (JPG, PNG, GIF).

## **Obtener la URL de la imagen:**

* Una vez que la imagen haya sido subida correctamente, se generará un enlace.
* Puedes copiar la URL desde el campo habilitado para utilizarla en otros apartados.
