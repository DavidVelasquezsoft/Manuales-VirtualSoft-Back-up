# Solución de Problemas

### Errores Comunes

Aquí se listan los problemas más frecuentes y sus posibles soluciones.&#x20;

| Problema                                                        | Posible causa                                                                                                      | Solución                                                                                                                                           |
| --------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| No puedo iniciar sesión.                                        | Contraseña incorrecta.                                                                                             | Comuníquese con soporte técnico                                                                                                                    |
| No se visualiza la funcionalidad deseada.                       | Usuario sin permisos.                                                                                              | Comuníquese con soporte técnico                                                                                                                    |
| No se aplican los cambios configurados.                         | Valores inválidos en los campos.                                                                                   | Verifique que los datos ingresados sean correctos y estén dentro del rango permitido.                                                              |
| La página no carga correctamente.                               | Problema con el navegador o [caché](https://app.gitbook.com/o/QcwavWzh0dfIwPyknoIT/s/mbqa0WvDWam8G20QQoIZ/#cache). | Intente borrar la [caché](https://app.gitbook.com/o/QcwavWzh0dfIwPyknoIT/s/mbqa0WvDWam8G20QQoIZ/#cache) del navegador o pruebe con otro navegador. |
| Se muestra un mensaje de error al intentar guardar los cambios. | Campos obligatorios vacíos o formato de datos incorrecto.                                                          | Revise que todos los campos requeridos estén completos y con valores válidos.                                                                      |
| Se cerró sesión inesperadamente.                                | Expiración de sesión o problema en la autenticación.                                                               | Intente iniciar sesión nuevamente y revise si el problema persiste, si es así comuníquese con soporte                                              |
| No se puede acceder a la plataforma.                            | Problema de conexión a internet o mantenimiento del sistema.                                                       | Verifique su conexión y consulte con el equipo de soporte si hay mantenimiento en curso.                                                           |
| Los filtros no devuelven resultados.                            | No hay datos disponibles o error en la consulta.                                                                   | Pruebe con otros filtros y asegúrese de que existan registros.                                                                                     |
| No se pueden editar los datos del usuario.                      | Restricciones de permisos o error en la base de datos.                                                             | Verifique los permisos del usuario y consulte con soporte si persiste el problema.                                                                 |
