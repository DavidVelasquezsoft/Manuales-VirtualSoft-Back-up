# Cajeros

<figure><img src="../../.gitbook/assets/image (353).png" alt=""><figcaption><p>Figura#1: Captura de pantalla seccion cajeros.</p></figcaption></figure>

Podrás buscar información sobre los cajeros utilizando los siguientes filtros de búsqueda:

* **ID Usuario:** Identificador único del cajero.
* **Nombre:** Nombre del cajero.
* **Login:** Nombre de usuario con el que el cajero accede al sistema.
* **País:** País donde está registrado el cajero.
* **Punto de Venta:** Debes seleccionar el punto de venta asociado al cajero.

Una vez aplicada la búsqueda, en la parte inferior se desplegará una tabla con la siguiente información:

* **ID Usuario:** Identificador único del cajero.
* **Nombre:** Nombre del cajero.
* **Login:** Nombre de usuario con el que accede al sistema.
* **Moneda:** Tipo de moneda con la que opera el cajero.
* **ID Punto de Venta:** Identificador del punto de venta al que está asignado el cajero.
* **Nombre Punto de Venta:** Nombre del punto de venta asociado al cajero.
