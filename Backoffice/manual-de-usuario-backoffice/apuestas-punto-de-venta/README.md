# Apuestas Punto de venta

Este modulo solo esta disponible para efectuar una apuesta, se solicitará el número de documento (DNI) del usuario, con el cual se realizará la operación en la plataforma.

{% hint style="warning" %}
**Nota:** Para realizar la apuesta el usuario debe estar previamente registrado en la plataforma, salvo cuando se habilita el modo de _apuestas_ anónimas. En este modo, según la configuración establecida por el partner _/_ país definidas en [**Ajustes generales**](/broken/pages/y9CSJzuItyKui5k9v8xg), se podrán requerir o no ciertos datos del usuario.
{% endhint %}
