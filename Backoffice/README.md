# Manual de usuario Backoffice

### **Introducción:**

En este manual se detallarán las diferentes funcionalidades y herramientas disponibles dentro de la plataforma **Backoffice**. Esta plataforma funciona como un sistema de gestión centralizado que permite administrar, supervisar y configurar distintos aspectos operativos de las plataforma, proporcionando control sobre usuarios, transacciones, promociones, reportes y otros módulos clave.

### **Objetivo:**

El objetivo de este manual es proporcionar una guía clara y detallada sobre el uso de cada una de las funcionalidades dentro del **Backoffice**, permitiendo a los usuarios operar de manera eficiente la plataforma. Se explicará la navegación, configuración y gestión de los distintos módulos, asegurando que el usuario pueda realizar sus tareas de manera óptima y sin dificultades.
