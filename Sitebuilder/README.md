# Manual de usuario

_**Versión 2**_

## Introducción:

Este manual de usuario tiene como propósito brindar una guía detallada sobre la funcionalidad de la plataforma en la creación, edición y administración desde **Site Builder** disponible tanto para dispositivos móviles como para PC. A través de esta documentación, los usuarios podrán familiarizarse con las herramientas y opciones disponibles, facilitando la gestión eficiente del sitio web.

## **Objetivo**

Proporcionar instrucciones claras y estructuradas para que los usuarios puedan utilizar el **Site Builder** de manera óptima, permitiéndoles personalizar, actualizar y administrar su sitio web de forma intuitiva y sin complicaciones.
