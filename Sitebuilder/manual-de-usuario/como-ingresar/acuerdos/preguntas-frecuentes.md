# Preguntas frecuentes

<figure><img src="../../../.gitbook/assets/image (230).png" alt=""><figcaption><p>Figura#1: Captura de pantalla sección preguntas frecuentes.</p></figcaption></figure>

## ¿Cómo agregar una nueva categoría de preguntas?

Haz clic en el botón verde **Agregar** ubicado en la parte superior derecha de la pantalla.

Se desplegará una tarjeta donde deberás completar los siguientes datos:

* **Ruta de Preguntas**: Especifica la ruta URL interna que se asociará a esta categoría. Por ejemplo: `/home`.
* **Título**: Indica el nombre que llevará la categoría. Ejemplo: `¿Cómo registrarse?`

<figure><img src="../../../.gitbook/assets/image (231).png" alt=""><figcaption><p>Figura#2: Captura de pantalla agregar.</p></figcaption></figure>

## ¿Cómo agregar un subtítulo o subcategoría?

Dentro de la tarjeta creada, haz clic en el ícono **Agregar** (ícono de “+” en verde).

Se desplegará un nuevo apartado donde deberás completar:

* **Ruta de Preguntas**: Indica la URL específica para esta subcategoría. Ejemplo: `/new-casino`.
* **Subtítulo Preguntas**: Ingresa el nombre del subtítulo. Ejemplo: `Casinos Nuevos`.

<figure><img src="../../../.gitbook/assets/image (232).png" alt=""><figcaption><p>Figura#3: Captura de pantalla agregar subtitulo.</p></figcaption></figure>

#### ¿Cómo agregar el contenido de la pregunta?

Dentro de cada subcategoría, haz clic en el ícono de **Texto** (ícono de engranaje⚙️).

Se abrirá un **pop-up** donde podrás escribir el contenido explicativo de la pregunta o sección.

Este texto se mostrará directamente en la plataforma, y puede incluir código HTML si deseas aplicar formatos personalizados.

<div align="center"><figure><img src="../../../.gitbook/assets/image (233).png" alt="" width="563"><figcaption><p>Figura#4: Captura de pantalla configurar texto.</p></figcaption></figure></div>

#### Funciones adicionales

* **Ordenar**: Puedes modificar el orden de aparición de las secciones o preguntas usando el ícono de flecha.
* **Eliminar**: Usa el ícono de papelera para eliminar secciones o preguntas.
* **Buscar**: Utiliza la barra de búsqueda para encontrar preguntas por texto original o personalizado.
