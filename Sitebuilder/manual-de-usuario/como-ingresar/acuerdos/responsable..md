# Responsable.

Esta será la vista inicial de esta sección, aquí podrás configurar las políticas de juego responsable en la plataforma.

<figure><img src="../../../.gitbook/assets/image (228).png" alt=""><figcaption><p>Figura #1: Captura de pantalla Juego responsable.</p></figcaption></figure>

Desde este apartado podrás realizar búsquedas de las políticas de juego responsable ya creados o crear términos y condiciones nuevos para la plataforma.

Utiliza el botón "**Agregar**" para crear nuevas políticas de juego responsable con el siguiente formulario:&#x20;

* **Ordenar**: Desde este botón podrás alternar entre las demás políticas de juego responsable creadas, para establecer el orden en el que se verán en la plataforma.
* **Ruta de juego responsable**: Agrega la ruta en la que irán las políticas de juego responsable.
* **Título**: Establece el título que tendrán las políticas de juego responsable.
* **Botón agregar**: Podrás agregar un submenú enlazado a las políticas de juego responsable principales, este botón desplegará los siguientes campos:
  * **Ruta de juego responsable**: Agrega la ruta en la que irán las políticas de juego responsable.
  * **Título**: Establece el título que tendrán las políticas de juego responsable.
  * **botón texto**: Se desplegará un editor de texto en el que podrás configurar el texto que tendrá el submenú de las políticas de juego responsable.
  * **Eliminar**: Este botón eliminará el submenú creado.
* **Botón eliminar**: Eliminarás las políticas de juego responsable creadas.
* **Botón desplegar**: Al seleccionar este botón, se desplegarán los submenús creados a partir de las políticas y privacidad principales.

Finalmente podrás completar las configuraciones con el botón "**Guardar**"
