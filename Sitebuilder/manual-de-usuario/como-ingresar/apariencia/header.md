# Header

Al acceder, podrás habilitar o deshabilitar los siguientes componentes:

<figure><img src="../../../.gitbook/assets/image (122).png" alt=""><figcaption><p>Figura#1: Captura de pantalla sección header.</p></figcaption></figure>

* **Reloj:**\
  Muestra u oculta el reloj en la interfaz, ubicado en una posición predeterminada. Al pasar el cursor sobre el ícono de información (¡), se mostrará una vista previa de cómo aparecerá en la plataforma.
* **Cambiar idioma:**\
  Permite que los usuarios seleccionen su idioma preferido. Al colocar el cursor sobre el ícono de información, se visualizará un ejemplo del componente en la plataforma.
* **Redes Sociales:**\
  Habilita o desactiva la visualización de los íconos que enlazan a las redes sociales. Al pasar el cursor sobre el ícono de información, se mostrará una vista previa.
* **Descargar App:**\
  Muestra u oculta un botón que facilita la descarga de la aplicación móvil. También incluye una vista previa al colocar el cursor sobre el ícono de información.
* **Franja Superior:**\
  Controla la visibilidad de la franja superior, comúnmente utilizada para mostrar notificaciones o mensajes destacados. Se incluye una vista previa al posicionar el cursor sobre el ícono de información.
* **Modo Claro/Oscuro:**\
  Activa o desactiva la visualización de esta funcionalidad que permite cambiar entre el tema claro y oscuro. Se puede visualizar un ejemplo pasando el cursor sobre el ícono de información.

En la parte inferior de este apartado, encontrarás un selector que permite definir el estilo de visualización del saldo dentro de la plataforma.

{% hint style="info" %}
Por ejemplo, si seleccionas el cuarto estilo, así es como se visualizará en la página.
{% endhint %}
