# Fondos

## Configuración de fondos

Esta interfaz permite gestionar los fondos visuales de diferentes secciones de la plataforma.

**Opciones disponibles:**

* **Fondo Home:** Imagen de fondo para la página principal.
* **Fondo Torneos:** Imagen de fondo para la sección de torneos.
* **Fondo Casino:** Imagen de fondo para la sección de casino.
* **Fondo Bingos:** Imagen de fondo para la sección de bingos.
* **Fondo Sorteos:** Imagen de fondo para la sección de sorteos.

## **Cómo cambiar un fondo:**

1. **Selecciona una categoría** de fondo (Home, Torneos, Casino, Bingos o Sorteos).
2. **Ingresa la URL de la imagen** en el campo de texto en la parte inferior, o selecciónala desde el dispositivo.
3. **Haz clic en el botón "Guardar"** para aplicar el nuevo fondo.

Este proceso actualizará la imagen de fondo de la categoría seleccionada en la plataforma.

{% hint style="warning" %}
**Nota:** Si no se guarda, los cambios no serán tomados en cuenta.
{% endhint %}
