# Redes sociales

## **Para agregar una nueva red social:**

1. Haz clic en el botón verde **“+”** ubicado en la parte superior.
2. Se añadirá una nueva tarjeta donde verás un menú desplegable con el texto **“Redes sociales”**.
3. Al hacer clic en este menú, se mostrarán las redes sociales disponibles para seleccionar (Facebook, Instagram, Tiktok, X, Youtube, Whatsapp).
4. Una vez seleccionada la red social, deberás ingresar la **URL correspondiente** en el campo de texto habilitado.
5. Finalmente, haz clic en el botón verde **“Guardar”** para registrar los cambios.
