# Subir imagen.

Al acceder a este apartado, encontrarás una interfaz que incluye una opción para subir imágenes, la cual generará un enlace.

<figure><img src="../../../.gitbook/assets/image (212).png" alt=""><figcaption><p>Figura#1: Captura de pantalla sección subir imagen.</p></figcaption></figure>

Para subir una imagen, se debe hacer clic en el botón “**cargar archivo**”. Al hacerlo, se abrirá un pop-up que te permitirá seleccionar la imagen deseada. Una vez seleccionada, aparecerá otro pop-up con la opción de copiar la URL de la imagen.
