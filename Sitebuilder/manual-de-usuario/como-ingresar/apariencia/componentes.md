# Componentes

Aquí tendrás disponible las siguientes configuraciones:

<figure><img src="../../../.gitbook/assets/image (156).png" alt=""><figcaption><p>Figura#1: Captura de pantalla sección Componentes.</p></figcaption></figure>

## **Banners compuestos:**

Puedes **agregar** o **editar banners compuestos**, los cuales permiten configurar varios elementos visuales con funcionalidad adicional.

<figure><img src="../../../.gitbook/assets/image (123).png" alt=""><figcaption><p>Figura#2: Captura de pantalla banners compuestos.</p></figcaption></figure>

* **Agregar Banner**: Sube una imagen en formato `.jpg` con un tamaño recomendado de 1920 px × 379 px.
* **Redirección**: Define la URL a la que se redirigirá al usuario al hacer clic en el banner.
* **Texto alternativo (Alt)**: Ingresa un texto descriptivo que se mostrará en caso de que la imagen no cargue, útil también para fines de accesibilidad.
* **Fechas de vigencia (Inicio y Fin)**: Establece el rango de fechas durante el cual el banner estará visible en la plataforma.

Estas configuraciones permiten personalizar la visualización de banners en función de campañas o eventos específicos.

## **Menú Iconos**:&#x20;

Desde esta opción podrás agregar o editar íconos a la plataforma.<br>

<figure><img src="../../../.gitbook/assets/image (158).png" alt="" width="563"><figcaption><p>Figura#3: Captura de pantalla Menú iconos.</p></figcaption></figure>

* **Título**: Debes agregar el título que tendrá el ícono.
* **Cargar** (📤): Al dar clic en este ícono podrás seleccionar la imagen que tendrá el ícono desde tu gestor de archivos.
* **Editar configuración (⚙️✏️)**:  Desde este ícono se desplegarán los siguientes campos para configurar el icono.
  * **ID:** Debes agregar el ID del ícono.
  * **Título**: Podrás modificar el título que tendrá el ícono.
  * **Menú URL**: Debes agregar la URL a la cual será redirigido el usuario al dar clic en el ícono.
  * **URL imagen de menú icono**: Debes agregar la URL de la imagen que tendrá el ícono.
  * **Sección**: Esta opción solo puede ser modificada por soporte técnico.
  * **Clases**: Esta opción solo puede ser modificada por soporte técnico.
  * **Editar color**: Esta opción solo puede ser modificada por soporte técnico.
  * **Nuevo**: Puedes establecer si el ícono tendrá un aviso indicando que es nuevo.
* **Botón Agregar**: Se agregará otro campo para configurar un nuevo ícono.

## **Cards Sportbook**:

Permite  **crear y administrar cards** para la sección Sportbook, vinculadas a diferentes categorías como deportes, eventos y ligas.<br>

<figure><img src="../../../.gitbook/assets/image (124).png" alt=""><figcaption><p>Figura#4: Captura de pantalla Cards Sportbook.</p></figcaption></figure>

### **Pasos para agregar una nueva Card:**

* **Selecciona la categoría** desde el menú lateral izquierdo:
  * Deportes
  * Eventos
  * Ligas

Haz clic en el botón verde “+” para agregar una nueva tarjeta. Al hacerlo, se mostrará un formulario con los siguientes elementos:

* **Subir imagen**: Haz clic en el ícono "+" para cargar una imagen desde tu dispositivo. Esta imagen representará visualmente la tarjeta.
* **ID:** Ingresa el número de identificación correspondiente. Este dato es esencial para identificar de forma única la tarjeta dentro del sistema. Corresponde al ID del juego en la plataforma; por ejemplo, el fútbol tiene asignado el ID **66**.

Para **guardar la tarjeta**, asegúrate de completar los campos obligatorios. Puedes **cerrar o cancelar** la creación de una Card haciendo clic en el botón con ícono de ❌ en la parte superior derecha del recuadro.

## **Cards imagen**:

Al hacer clic en el botón verde con el ícono **“+”**, se desplegará una tarjeta con los siguientes elementos:

<figure><img src="../../../.gitbook/assets/image (125).png" alt=""><figcaption><p>figura#5: Captura de pantalla card imagen</p></figcaption></figure>

* **Cargar imagen:**\
  Haz clic en el ícono **“+”** para seleccionar y subir una imagen desde tu dispositivo. Esta imagen representará visualmente el contenido de la tarjeta.
* **Título:**\
  En el campo **“Agregar título”**, escribe el nombre o título que deseas asignar a la tarjeta. Este texto servirá como una breve descripción o encabezado visible del contenido.
* **Opciones adicionales:**
  * (⚙️) Permite acceder a configuraciones adicionales relacionadas con la tarjeta.
    * **URL de imagen:**\
      Ingresa la URL de la imagen si prefieres cargarla desde una fuente externa en lugar de subirla desde tu dispositivo.
    * **URL de redirección:**\
      Especifica la dirección web a la que será redirigido el usuario al hacer clic en la imagen.

&#x20;(❌) permite eliminar la tarjeta actual si ya no se necesita.
