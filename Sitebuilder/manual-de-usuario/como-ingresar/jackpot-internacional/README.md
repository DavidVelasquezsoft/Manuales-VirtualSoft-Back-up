# Jackpot internacional

### **1. Acceso al Módulo**

**Ruta:** Site Builder > Jackpot

***

### 🧐 2. ¿Qué es un jackpot internacional?

> Premio acumulado de alcance internacional que incrementa su valor con cada participación de usuarios proveniente de múltiples países o plataformas. Se otorga cuando se cumplen las condiciones configuradas en el sistema para todas las operaciones participantes.

***

### 📂 3. Contenido del módulo

A continuación, encontrarás las subpáginas que detallan cada funcionalidad:<br>

* **Gestión:** permite consultar informes para monitorear su comportamiento y rendimiento y gestionar un jackpot ya sea que este activo o inactivo.

[gestion.md](gestion.md)
{% endcontent-ref %}

***

### 📌 4. Notas importantes

* **El pozo es compartido:** Todas las plataformas y países vinculados al jackpot aportan al mismo fondo acumulado bajo una misma moneda.
* **Los permisos determinan capacidades:** Solo administradores autorizados pueden modificar reglas, reiniciar valores o generar reportes oficiales del jackpot.
