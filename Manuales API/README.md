# Manuales API

### Manual API integración de casino

Permite a proveedores de juegos y equipos técnicos de operadores que necesiten interactuar con la API para lanzar juegos, realizar transacciones, validar sesiones y gestionar bonificaciones.

{% file src=".gitbook/assets/Manual_API_Casino.pdf" %}

### Manual API integración de pasarelas de depósitos

Permite realizar el proceso de integración con la API de VirtualSoft para la creación y gestión de solicitudes de depósito.

Está dirigida a equipos técnicos de agregadores, proveedores y operadores que necesiten interactuar con nuestros servicios mediante integración REST

{% file src=".gitbook/assets/Manual_API_Solicitud_De_Depósitos.pdf" %}

### Manual API consulta de bonos

Permitir la consulta, gestión y visualización de información relacionada con bonos asignados a los usuarios, centralizando los datos provenientes de diferentes fuentes internas del sistema.

{% file src=".gitbook/assets/Manual_API_Consulta_Bonos.pdf" %}

### Manual Recargas operador

La API de Redes Aliadas permite a puntos físicos autorizados realizar depósitos, retiros, reversión de transacciones y consultas en tiempo real mediante una integración segura con el sistema central. Este manual describe los endpoints, estructura de mensajes y lineamientos necesarios para su correcta implementación.

{% file src=".gitbook/assets/Manual_API_Recargas_Operator (7).pdf" %}

### Manual Registro Whatsapp

Esta documentación describe el uso de la API de Registro de Usuarios de VirtualSoft, diseñada para facilitar la incorporación de nuevos clientes. La API permite registrar usuarios de forma segura, validando su información básica y asignando promociones desde un punto de venta autorizado.

{% file src=".gitbook/assets/Manual_API_Registro_WhatsApp_VirtualSoft 1 (5).pdf" %}

### Manual Jugadores en tiempo real

Este manual describe el uso y la integración de la API de Información de Jugadores en Tiempo Real, desarrollada para permitir la consulta ágil y segura de datos generales de jugadores mediante un agente de inteligencia artificial

{% file src=".gitbook/assets/Manual_API_Jugadores_en_tiempo_real (4).pdf" %}

### Manual consultas de retiros y depósitos de un usuario.

Esta Api permite consultar los depósitos y retiros de un usuario de las diferentes operaciones de Virtualsoft por país, además permite filtros por fechas.

{% file src=".gitbook/assets/Manual_API_consultas-de-retiros-y-depósitos-de-un-usuario (3).pdf" %}

### Manual Funciones y modo de uso API GATEWAY

Esta guía está dirigida a los desarrolladores de casino que necesiten interactuar con el proyecto ApiGateway para realizar transacciones, validar cuerpos de petición por medio de hash y unificar sus desarrollos en un único punto de acceso.

{% file src=".gitbook/assets/Funciones y modo de uso API GATEWAY.pdf" %}
