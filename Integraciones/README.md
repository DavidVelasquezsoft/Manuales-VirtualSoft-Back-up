# Manual integraciones por proveedor.

## Introducción:&#x20;

Este manual te guiará paso a paso en el acceso y configuración de cada integración con proveedores agregados, desde la plataforma de Torneos y Bonos hasta su visualización en Usuarios Online.

## Objetivo:

Brindar una guía clara y detallada para gestionar y configurar las integraciones con proveedores en la plataforma, asegurando una correcta implementación y visualización de los datos.
