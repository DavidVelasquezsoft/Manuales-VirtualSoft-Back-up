# 3. Objetivo General

Implementar un modelo dinámico para la validación de APIs de casino mediante el uso\
de la herramienta “postman”, que permita verificar el correcto funcionamiento de los\
endpoints mediante la configuración flexible de métodos de autenticación, tipos de\
contenido, métodos HTTP, y casos de uso, garantizando la integridad, confiabilidad,\
disponibilidad del servicio expuesto por la API.
