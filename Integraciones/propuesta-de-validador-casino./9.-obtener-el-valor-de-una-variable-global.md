# 9. Obtener el valor de una variable global

Capturar el valor en una variable javascript: podemos obtener el valor de las variables\
para utilizarlas en la validación de un caso de prueba.

```javascript
Var balance = pm.globals.get("balance");
```

Para usar la variable simplemente se la invoca mediante la siguiente instrucción:\
`{{nombre_variable}}`
