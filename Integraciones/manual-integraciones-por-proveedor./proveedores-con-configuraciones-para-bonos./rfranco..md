# RFRANCO.

Esta es la visual completa del Bono FreeSpin

<figure><img src="../../.gitbook/assets/image (90).png" alt="" width="563"><figcaption><p>Figura #1: Captura de pantalla de la sección FreeSpin.</p></figcaption></figure>

En este caso nos céntratenos solo en las configuraciones necesarias para la configuración de los nuevos juegos integrados por el proveedor RFRANCO.

* **Rango de fechas:** El botón **"Agregar"**, desplegará una opción que te permitirá configurar las fechas durante las cuales el bono estará activo. Aquí deberás seleccionar tanto la **fecha inicial** como la **fecha final** para definir el período de vigencia del bono.

{% hint style="warning" %}
**Nota**: El rango de fechas no puede ser menor a 1 día.
{% endhint %}

* **Botón proveedores:** Este botón Desplegará un listado solo con los proveedores que tienen el Bonus System activo, en este caso se debe visualizar y seleccionar la opción "R**FRANCO"**.
* **Botón Productos:** Este botón despegará 2 listas para seleccionar un juego dependiendo el proveedor, en este caso, es necesario volver a seleccionar el proveedor "R**FRANCO**", para visualizar los juegos disponibles.
* **Moneda**: Para poder dar uso a los bonos FreeSpin con el proveedor RFRANCO, es necesario seleccionar la moneda del país con el que estamos generando el bono, esto activará las siguientes configuraciones.
  *   **Valor por ronda:** Este es el monto que tendrá el bono por cada ronda otorgada.

      <div data-gb-custom-block data-tag="hint" data-style="warning" class="hint hint-warning"><p><strong>Nota</strong>: Para ingresar el valor por ronda, primero es necesario validar en los juegos el monto de apuesta disponible para apostar.</p></div>
  * **Rondas gratuitas:** En este campo podrás ingresar la cantidad de rondas gratuitas que el usuario podrá jugar con este bono.
  * **Jugadores:** En esta configuración, se debe ingresar el ID de cada usuario que tendrá el bono activo. Este proceso requiere cargar un archivo en formato **CSV** con los ID´s correspondientes. El sistema tomará los datos del archivo y activará automáticamente el bono para los jugadores especificados.

Estas son las configuraciones principales y necesarias para generar un bono FreeSpin con los juegos del proveedor RFRANCO, en caso de necesitar información más detallada sobre cómo crear el bono FreeSpin, puedes acceder a la siguiente página:
