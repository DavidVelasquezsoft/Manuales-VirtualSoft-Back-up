# KAGAMING.

Esta es la visual completa del Bono FreeSpin

<figure><img src="../../.gitbook/assets/image (90).png" alt="" width="563"><figcaption><p>Figura #1: Captura de pantalla de la sección FreeSpin.</p></figcaption></figure>

En este caso nos céntratenos solo en las configuraciones necesarias para la configuración de los nuevos juegos integrados por el proveedor KAGAMING.

* **Rango de fechas:** El botón **"Agregar"**, desplegará una opción que te permitirá configurar las fechas durante las cuales el bono estará activo. Aquí deberás seleccionar tanto la **fecha inicial** como la **fecha final** para definir el período de vigencia del bono.

{% hint style="warning" %}
**Nota:** La fecha final no puede superar los 3 meses desde la fecha de inicio.
{% endhint %}

* **Botón proveedores:** Este botón Desplegará un listado solo con los proveedores que tienen el Bonus System activo, en este caso se debe visualizar y seleccionar la opción "**KAGAMING"**.
* **Botón Productos:** Este botón despegará 2 listas para seleccionar un juego dependiendo el proveedor, en este caso, es necesario volver a seleccionar el proveedor "**KAGAMING**", para visualizar los juegos disponibles.

{% hint style="warning" %}
**Nota**: Al configurar los campos "**Proveedores**" y "**Productos**" se desplegará una tabla en la cual debes especificar el porcentaje del bono que asumirá el proveedor.
{% endhint %}

*   **Moneda**: Para poder dar uso a los bonos FreeSpin con el proveedor KAGAMING, es necesario seleccionar la moneda del país con el que estamos generando el bono, esto activará las siguientes configuraciones.

    * **Valor por ronda:** Este es el monto que tendrá el bono por cada ronda jugada. El valor por ronda no corresponde al monto de apuesta del juego, sino al número de orden en el que se encuentra dicho valor.
    * **Rondas gratuitas:** En este campo podrás ingresar la cantidad de rondas gratuitas que el usuario podrá jugar con este bono.
    * **Jugadores:** En esta configuración, se debe ingresar el ID de cada usuario que tendrá el bono activo. Este proceso requiere cargar un archivo en formato **CSV** con los ID´s correspondientes. El sistema tomará los datos del archivo y activará automáticamente el bono para los jugadores especificados.

    <div data-gb-custom-block data-tag="hint" data-style="info" class="hint hint-info"><p>Ejemplo: Para el campo "Valor por ronda", si los montos disponibles en el juego son:</p><ol><li>20/S</li><li>15/S</li><li>30/S</li></ol><p>Si deseas seleccionar el valor 15, debes ingresar <strong>2</strong>, ya que corresponde a su posición en la lista.</p></div>
