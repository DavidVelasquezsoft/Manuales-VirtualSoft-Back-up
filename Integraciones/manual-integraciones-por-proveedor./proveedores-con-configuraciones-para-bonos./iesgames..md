# IESGAMES.

Esta es la visual completa del Bono FreeSpin

<figure><img src="../../.gitbook/assets/image (90).png" alt="" width="563"><figcaption><p>Figura #1: Captura de pantalla de la sección FreeSpin.</p></figcaption></figure>

En este caso nos céntratenos solo en las configuraciones necesarias para la configuración de los nuevos juegos integrados por el proveedor IESGAMES.

* **Rango de fechas:** El botón **"Agregar"**, desplegará una opción que te permitirá configurar las fechas durante las cuales el bono estará activo. Aquí deberás seleccionar tanto la **fecha inicial** como la **fecha final** para definir el período de vigencia del bono.
* **Botón proveedores:** Este botón Desplegará un listado solo con los proveedores que tienen el Bonus System activo, en este caso se debe visualizar y seleccionar la opción "**IESGAMES"**.
  * **Prefijo**: Al seleccionar el proveedor IESGAMES se desplegará este campo en el cual debes ingresar dos dígitos, ya sean números o letras. Este será el inicio del código promocional.
* **Botón Productos:** Este botón despegará 2 listas para seleccionar un juego dependiendo el proveedor, en este caso, es necesario volver a seleccionar el proveedor "**IESGAMES**", para visualizar los juegos disponibles.

{% hint style="warning" %}
**Nota**: Al configurar los campos "**Proveedores**" y "**Productos**" se desplegará una tabla en la cual debes especificar el porcentaje del bono que asumirá el proveedor.
{% endhint %}

*   **Moneda**: Para poder dar uso a los bonos FreeSpin con el proveedor IESGAMES, es necesario seleccionar la moneda del país con el que estamos generando el bono, esto activará las siguientes configuraciones.

    * **Numero de cartones**: En este campo debes ingresar la cantidad de cartones gratuitos que se darán a los usuarios.

    <div data-gb-custom-block data-tag="hint" data-style="warning" class="hint hint-warning"><p><strong>Nota:</strong> Los cartones tienen el valor de 1/S por defecto (la moneda puede variar, pero el monto se mantiene fijo)</p></div>

    * **Jugadores:** En esta configuración, se debe ingresar el ID de cada usuario que tendrá el bono activo. Este proceso requiere cargar un archivo en formato **CSV** con los ID´s correspondientes. El sistema tomará los datos del archivo y activará automáticamente el bono para los jugadores especificados.

{% hint style="warning" %}
**Nota:** Al crear el bono, se genera una lista de códigos promocionales con la cantidad de jugadores ingresada previamente en el apartado "**Jugadores**". Es responsabilidad de la persona que crea el bono enviar estos códigos a los usuarios.
{% endhint %}
